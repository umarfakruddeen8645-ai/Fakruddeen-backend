# Fakruddeen Backend

Backend mai tsaro da aka gina da **Node.js + Express + PostgreSQL**, tare da Docker deployment da automated tests.

---

## 🚀 Features
- User authentication (JWT + bcrypt)
- Task management (CRUD)
- AI modules (speech recognition da face detection)
- Security checks (anomaly detection, encryption)
- Self-improvement module (auto-code generation)
- Remote troubleshooting diagnostics
- Docker + docker-compose deployment
- Automated testing script
